# Getting started
1. **Copy .env and adjust env vars if necessary**

```bash
$  cp .env.example .env.local
```

2. Install dependencies.

```bash
$  yarn i --frozen-lockfile
```

3. Start dev server with hot reload at http://localhost:3000.

```bash
$  yarn run dev
```
